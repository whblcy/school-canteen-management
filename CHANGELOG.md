# 更新日志

## [v1.0] - 2026-06-20

### 功能特性

- **概览统计** - 食材总数、库存总值、低库存预警数实时展示
- **提醒中心** - 低库存提醒、过期预警、已过期食材集中管理
- **财务统计** - 月度收支分析、年度趋势图、分类占比、供应商占比
- **食材管理** - 增删改查、Excel 导入导出、批量操作
- **入库管理** - 入库记录、库存自动增加、总价自动计算
- **出库管理** - 出库记录、库存自动扣减、库存不足拦截
- **库存盘点** - 系统库存 vs 实际库存、差异计算、自动校正
- **供应商管理** - 增删改查、联系方式管理
- **分类管理** - 增删改查、默认 8 大分类
- **系统设置** - 数据备份/恢复、修改密码
- **操作日志** - 全操作记录、便于审计追溯
- **用户登录** - 默认管理员账号、SHA256+盐值密码安全

### 技术特性

- macOS 风格 UI 设计
- SQLite3 本地数据库存储
- 上下文管理器保证数据库连接安全关闭
- SQL 注入防护（字段白名单校验）
- 事务保护（BEGIN/COMMIT/ROLLBACK）
- 运行日志系统（logs/目录）
- 全面自动化测试（51 项测试覆盖 12 个模块）

### 默认登录

- 用户名: `admin`
- 密码: `admin123`

### 系统要求

- Windows 10/11 64位
- 屏幕分辨率建议 1920x1080 及以上
